3. svg-circle-progress
参数说明

### progress
进度值 数字 例如0.7

### progressOption 
进度条相关参数
#### radius 内圆半径
#### strokeWidth 画笔宽度 即圆弧宽度
#### backColor 底色
#### startColor 渐变色开始值 数组 [a,b,c] 与rgb颜色对于
#### endColor 渐变色结束值 同上
#### durtion 动画时长
#### step 步数 设置越大渐变效果越好，但是会带来相应的性能损耗，默认100，一般情况下无需修改
